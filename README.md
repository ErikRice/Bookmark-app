# Welcome to Bookmark-app!

This app allows you to store your favorite websites and their links in one place.

## How To:

Add the name of your website in the name input. Next, add the site's URL link in the URL input. (Ex. `https://your-favorite-site.com`) Once you've typed everything in, hit the submit button. It should now show a link to your site down below. If you would like to delete a site from the list, select the delete button underneath the site's name. 
Happy Bookmarking!
